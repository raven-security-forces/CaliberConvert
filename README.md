# CaliberConvert
A collection of alternative configurations for the Arma 3 unit Raven Security Forces

## What it does
Makes changes to various weapons in terms of weight and caliber to fall in line with RSF standards. 

## Current Weapons
 	KH-2002
	MG5
	MSBS-5.56B
	QBU-88
	RFB
	Stoner 99
